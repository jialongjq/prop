- OBJECTE DE LA PROVA

Comprovar el funcionament de la classe usuari.
  
- FITXER DE PROVES OPCIONAL

DATA/usuari.txt

- VALORS ESTUDIATS
El fitxer "usuaris.txt" inclou l'id d'usuari, el nom d'usuari, la cotrasenya d'usuari, boolea que especifica si aquest usuari es administrador, nou nom de l'usuari i una nova contrasenya. Utilitzant la caixa blanca.

- EFECTE ESTUDIATS

Constructures, Getters i Setter de la classe Usuari.

- OPERATIVA

Inici: carregar els valors del fitxer, cal escollir una de les opcions propocionades:
  	"1" -> Testejar Constructura
  	"2" -> Testejar Getters
  	"3" -> Testejar Setters
  	"4" Sortir del driver
  	
Testejar Constructura: crea un usuari amb id d'usari, nom d'usuari, la contrasenya d'usuari, boolea indicant si es administrador. 
  
  	
Testejar Getters: cal escollir una de les opcions propocionades:
  	         "1" ->  GetIdUsuari
                "2" -> GetNomUsuari
                "3" -> GetContrasenyaUsuari
                "4" -> GetEsAdmin
  
Cada funcio per comprovar getter, crea un usuari amb la informacio donada, i ens propociona el valor esperat d'usuari amb el valor real obtingut.
  
Testejar Setters: cal escollir una de les opcions propocionades:
                "1" -> SetNomUsuari
                "2" -> SetContrasenyaUsuari
  	
Cada funcio per comprovar setter, crea un usuari amb la informacio donada, fa un setter, i ens propociona el valor esperat d'usuari amb el valor real obtingut.
